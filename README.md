# さんごう防災3Dマップ

奈良県三郷町の [PLATEAU 3D都市モデル (2025年度)](https://www.geospatial.jp/ckan/dataset/plateau-29343-sango-cho-2025) を活用した防災Webアプリです。

## 解決する自治体課題

三郷町は大和川と竜田川の合流部に市街地が広がり、**洪水浸水リスク**を抱える一方、信貴山麓には**土砂災害警戒区域**が多数指定されています。紙のハザードマップでは「自分の家がどのくらい浸かるのか」が直感的に伝わりにくいという課題があります。

本アプリは PLATEAU の3D建物モデルにハザード情報を重ね、住民がリスクを「自分ごと」として理解できるようにします。

## 機能

| 機能 | 内容 |
|---|---|
| 3D都市モデル表示 | PLATEAU建築物モデル (3D Tiles) + PLATEAU地形を表示 |
| ハザード重畳 | 洪水浸水想定区域 (想定最大規模)・土砂災害警戒区域3種を切替表示・不透明度調整 |
| 指定緊急避難場所 | 国土地理院データから町内の避難場所を表示 (対応災害種別つき) |
| 地点リスク診断 | 地図クリックでその地点の浸水深ランク・土砂災害区域該当・最寄り避難場所と距離を表示 |
| 浸水疑似体験 | スライダーで仮想水面 (0〜10m) を3D表示し、建物と浸水深を見比べ |

## アーキテクチャ

- **Vite + CesiumJS** による静的サイト。サーバー不要で GitHub Pages 等にそのままデプロイ可能
- PLATEAUデータは同梱せず、**実行時に公式配信サービスから取得**
  - 建築物3D Tiles: [PLATEAUデータカタログAPI](https://github.com/Project-PLATEAU/plateau-streaming-tutorial) (`/datacatalog/plateau-datasets`) から市区町村コード `29343` で動的解決 (URLハードコードなし・localStorageに7日キャッシュ)
  - ハザードタイル: 国土地理院 重ねるハザードマップ配信タイル
  - 避難場所: 国土地理院 指定緊急避難場所 GeoJSONタイル (skhb01〜08)
- 地点診断はハザードタイルのピクセル色を canvas で読み取り、公式凡例色と照合して浸水深ランクを判定

## 開発・実行

```bash
npm install
npm run dev      # 開発サーバー
npm run build    # dist/ に静的ビルド
```

`main` 系ブランチへのプッシュで GitHub Pages へ自動デプロイするワークフローを同梱しています (`.github/workflows/deploy.yml`、リポジトリ設定で Pages のソースを「GitHub Actions」にしてください)。

## 既知の制約

- 開発環境のネットワーク制限により、PLATEAU配信サービス・地理院タイルへの実接続テストは未実施です。初回起動時は画面左下「データ読み込み状況」で各データの取得状態を確認してください
- PLATEAU-Terrain は公式チュートリアル掲載の公開Ionトークンを使用しています。失効した場合は平坦表示に自動フォールバックします
- 本アプリの表示は参考情報です。避難の判断は三郷町の公式情報に従ってください
