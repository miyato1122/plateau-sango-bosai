    オフライン利用

    通信できない災害時に備えて、町内の地図とハザードデータを端末に保存できます (約10〜20MB)。保存後は電波がなくても地点診断と避難場所の確認ができます。

    データ取得状況

      出典

      • 3D都市モデル (Project PLATEAU)・避難施設等 / 国土交通省
      • 地理院タイル (淡色地図・空中写真・標高タイル) / 国土地理院 — 標高タイルを加工して3D浸水表示を作成
      • PLATEAU-Terrain (3D地形) / 国土交通省 — 地形データは、測量法に基づく国土地理院長承認（使用）R3JHs 778を得て使用
      • 重ねるハザードマップ (洪水浸水想定・土砂災害警戒区域) / 国土地理院 — 配信タイルを解析してリスク判定・面積集計に利用
      • 指定緊急避難場所データ・地名検索API / 国土地理院
      • 地形表示: CesiumJS (Apache License 2.0)
      • 安全ルートの道路網データ (同梱時): © OpenStreetMap contributors (ODbL)

      PLATEAU配信サービス・地理院タイルは試験的提供を含み、提供の継続やサービス品質は保証されません。本アプリの表示はあくまで参考情報です。実際の避難の判断は、三郷町の公式ハザードマップおよび避難情報に従ってください。
